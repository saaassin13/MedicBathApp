# Bug #002：主页缺少底部导航栏和滑动

**现象**：选择奶厅进入主页后，页面不能滑动，下方导航栏也没有显示。

**根因**：路由配置中所有页面都是平级路由，没有使用 `ShellRoute` 包裹需要底部导航栏的页面（首页、数据、记录、我的）。虽然项目已有 `BottomNavScaffold` 组件，但路由层未集成。

**修复方案**：
1. 使用 `ShellRoute` 将 4 个 Tab 页面（home、data、records、personal-center）包裹
2. 在 `ShellRoute` 的 `builder` 中使用 `BottomNavScaffold`
3. 其他独立页面（select-farm、fault-log、maintenance-plan、maintenance-record、settings）保持平级

**验证方法**：
1. 登录 → 选择奶厅 → 进入首页，底部应显示导航栏
2. 导航栏应高亮"首页"Tab
3. 页面内容应可上下滚动
4. 点击"数据"、"记录"、"我的"应能切换页面并更新导航栏高亮
