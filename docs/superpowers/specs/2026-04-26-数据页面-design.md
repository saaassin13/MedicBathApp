## 1. 页面概述

| 项目 | 内容 |
|-----|------|
| **页面定位** | 设备运行数据的详细曲线展示页面 |
| **核心功能** | 作业数据图表展示、药液用量统计、覆盖异常分析 |

## 6. 数据结构

### 数据页面数据模型

```typescript
interface DataPageData {
  // 位置信息
  location: {
    farmName: string;      // "双城牧场"
    period: string;       // "一期奶厅"
    deviceName: string;   // "一号转盘"
  };

  // 作业数据图表
  chartData: {
    dates: string[];                    // ["04/07", "04/08", ...]
    cowCounts: number[];               // 药浴牛数数组
    recognitionRates: number[];        // 乳头识别率数组
  };

  // 药液用量统计
  usageStats: {
    totalUsage: number;      // 总用量 (L)
    avgUsage: number;        // 日均用量 (ML)
    maxDailyUsage: number;   // 最大日用量 (ML)
    maxDailyDate: string;    // 最大日用量日期 "04/13"
  };

  // 覆盖异常统计
  coverageStats: Array<{
    type: string;           // 异常类型
    count: number;          // 数量
    percentage: number;     // 占比
  }>;

  // 覆盖异常类型枚举
  coverageTypes: [
    { name: "未脱杯", count: 50, percentage: 25 },
    { name: "防褪链", count: 30, percentage: 15 },
    { name: "牛腿过窄", count: 20, percentage: 10 },
    { name: "异常跳过", count: 15, percentage: 8 }
  ];
}
```

## 10. 备注

- 无设备切换按钮，当前页面显示选中奶厅的综合数据
- 图表采用双轴设计，左Y轴对数量（柱状），右Y轴对百分比（折线）
- 日期格式为MM/DD（如04/13）

### 10.1 适配优化项

| 项目 | 当前实现 | 优化建议 | 优先级 |
|-----|---------|---------|--------|
| 图表高度 | 固定 200px | 可根据屏幕高度动态计算 `MediaQuery.of(context).size.height * 0.3` | 低 |
| 字号 | 统一 12px | 可考虑增加系统字号偏好支持 | 低 |

**说明**：当前实现已通过基本适配测试，在主流Android/iOS设备上表现正常，优化项为可选增强。
