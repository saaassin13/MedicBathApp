## 1. 页面概述

| 项目 | 内容 |
|-----|------|
| **页面定位** | 查看历史药浴操作记录 |
| **核心功能** | 按日期分组展示记录列表、显示运行时长和核心指标 |
| **前置条件** | 用户已登录并选择奶厅 |
| **后置结果** | 可查看历史记录后跳转其他页面 |

### 3.3 记录卡片

#### 卡片结构

```
┌────────────────────────────────────────┐
│ 13:11:22 - 15:32:01    02小时20分钟15秒 │  ← 首行：时间区间 + 运行时长
│ 药浴:425头 │ 识别率:99.8% │ 均量:4.8ML │  ← 次行：三栏指标
└────────────────────────────────────────┘
```

#### 首行：时间与时长

| 元素 | 格式 | 说明 |
|-----|------|-----|
| 开始时间 | HH:mm:ss | 如 13:11:22 |
| 结束时间 | HH:mm:ss | 如 15:32:01 |
| 连接符 | " - " | 前后空格 |
| 运行时长 | XX小时XX分钟XX秒 | 中文格式，如 02小时20分钟15秒 |

#### 次行：三栏指标（同一行）

| 指标 | 格式 | 示例 |
|-----|------|-----|
| 药浴头数 | 药浴：数值 头 | 药浴：425 头 |
| 识别率 | 识别率：数值 % | 识别率：99.8 % |
| 均量 | 均量：数值 ML | 均量：4.8 ML |

#### 样式规范

| 属性 | 设计规范 |

| **卡片背景** | 白色 |
| **卡片圆角** | 8px |

| **指标标签** | 灰色 |
| **指标数值** | 橙色 #FF6600 |

| **布局** | 首行：时间左 + 时长右；次行：三栏均分 |

## 6. 数据结构

### 记录页面数据模型

```typescript
interface RecordPageData {
  // 位置信息
  location: {
    farmName: string;      // "双城牧场"
    period: string;       // "一期奶厅"
    deviceName: string;   // "一号转盘"
  };

  // 记录列表（按日期分组）
  records: Array<{
    date: string;                    // 日期分组标题 "今天"/"昨天"/"4月5日"
    items: Array<{
      startTime: string;           // 开始时间 "HH:mm:ss"
      endTime: string;              // 结束时间 "HH:mm:ss"
      duration: {                   // 运行时长
        hours: number;             // 小时
        minutes: number;          // 分钟
        seconds: number;          // 秒
      };
      cowCount: number;            // 药浴头数
      recognitionRate: number;     // 乳头识别率 %
      avgUsage: number;            // 药液均量 ML
    }>;
  }>;
}
```

### 运行时长格式化

```typescript
// 运行时长格式：XX小时XX分钟XX秒
formatDuration(seconds: number): string {
  const h = Math.floor(seconds / 3600);
  const m = Math.floor((seconds % 3600) / 60);
  const s = seconds % 60;
  return `${h.toString().padStart(2, '0')}小时${m.toString().padStart(2, '0')}分钟${s.toString().padStart(2, '0')}秒`;
}
```

## 10. 备注

- 无标签切换，当前默认显示药浴记录
- 日期居中显示，区分不同日期段落
- 运行时长采用中文格式（XX小时XX分钟XX秒）
- 记录按日期分组，便于用户追溯历史数据

### 10.1 适配优化项

| 项目 | 当前实现 | 优化建议 | 优先级 |
|-----|---------|---------|--------|
| 三栏指标布局 | spaceBetween 固定三等分 | 可改为 Flexible 防止极窄屏溢出 | 低 |

**说明**：当前实现已通过基本适配测试，在主流Android/iOS设备上表现正常，优化项为可选增强。
