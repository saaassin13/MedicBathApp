# 故障日志页面 - 设计规格文档

**日期**：2026-04-26
**页面**：故障日志页面
**类型**：业务功能 - 个人中心模块
**入口**：从"我的"页面跳转而来
**状态**：已确认

---

## 1. 页面概述

| 项目 | 内容 |
|-----|------|
| **页面定位** | 记录和跟踪设备故障 |
| **核心功能** | 展示故障列表、时间排序、三级颜色状态标识 |
| **前置条件** | 用户已登录并选择奶厅 |
| **进入来源** | 主要从"我的"页面进入，也可从首页异常报警条点击进入 |
| **后置结果** | 可查看故障详情或返回上一页 |

### 区域划分

| 区域 | 内容 | 说明 |
|-----|------|-----|
| 导航栏 | 奶厅选择器 + 下拉箭头▼ + 关闭按钮 | 橙色背景，奶厅靠左 |
| 列表卡片 | 头部图标+标题 + 故障流水 | 白色卡片，圆角8px |
| 故障项 | 时间戳 + [级别] + 描述 | 垂直排列 |

| 底部 | iOS系统Home指示条 | 无底部导航栏 |

### 导航说明

**全屏页面设计**：故障日志页面是全屏二级页面，**不显示底部导航栏**。

- 进入方式：从"我的"页面点击"故障日志"入口，或从首页异常报警条点击进入
- 导航栏：奶厅选择器（靠左）+ 关闭按钮（靠右）
- 返回行为：点击 X 按钮返回"我的"页面

## 6. 数据结构

### 故障日志数据模型

```typescript
interface FaultLogItem {
  timestamp: string;        // "03/07 11:49:55.099" 精确到毫秒
  level: "error" | "warning" | "info";  // 错误/警告/提示
  description: string;      // 故障描述
  deviceId?: string;        // 设备ID（可选）
  deviceName?: string;      // 设备名称（可选）
}

interface FaultLogPageData {
  // 位置信息
  location: {
    farmName: string;      // "双城牧场"
    period: string;       // "一期奶厅"
    deviceName: string;   // "一号转盘"
  };

  // 故障列表
  logs: FaultLogItem[];

  // 分页信息
  pagination: {
    page: number;
    pageSize: number;
    hasMore: boolean;
  };
}
```

### 状态映射

| level值 | 显示标签 | 颜色 |
|---------|---------|------|
| "error" | [错误] | 红色 #F44336 |
| "warning" | [警告] | 橙色 #FF9800 |
| "info" | [提示] | 绿色 #4CAF50 |
