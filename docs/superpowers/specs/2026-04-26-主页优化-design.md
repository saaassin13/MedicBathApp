# 主页优化设计

**日期**：2026-04-26
**状态**：已确认（v4 修订）

---

## 1. 概述

对主页（HomePage）进行 UI 优化调整，包括奶厅标题样式、告警卡片背景、监控卡片合并、状态卡片布局等。

---

## 2. 修改明细

### 2.1 奶厅标题（AppBar）

| 项目 | 修改前 | 修改后 |
|------|--------|--------|
| 字号 | 默认（约 20） | 14pt |
| 对齐 | 居中 | 靠左 |

### 2.2 告警卡片背景

| 状态 | 修改前 | 修改后 |
|------|--------|--------|
| 无异常（hasAlarm=false） | `AppColors.successGreen`（绿色） | 白色 |
| 有异常（hasAlarm=true） | `AppColors.errorRed`（红色） | 红色（不变） |

### 2.3 KPI 卡片

**自适应实现**：使用 `AspectRatio(aspectRatio: 1.6)` + `MediaQuery` 动态计算字号和间距，适配不同屏幕尺寸。

| 项目 | 修改前 | 修改后 |
|------|--------|--------|
| 布局 | 固定高度 | `AspectRatio(1.6)` 自适应 |
| 内边距 | `12px` | `screenHeight * 0.012` |
| 图标尺寸 | `16px` | `screenHeight * 0.016` |
| 标题字号 | `12px` | `screenHeight * 0.012` |
| 数值字号 | `24px` | `screenHeight * 0.024` |
| 单位字号 | `12px` | `screenHeight * 0.012` |
| 颜色 | `primaryOrange` | `textPrimary`（黑色） |

### 2.4 监控卡片

**自适应实现**：使用 `MediaQuery` 动态计算高度和字号，适配不同屏幕尺寸。

| 项目 | 修改前 | 修改后 |
|------|--------|--------|
| 监控画面高度 | `140px` | `screenHeight * 0.16` |
| 监控图标尺寸 | `48px` | `screenHeight * 0.04` |
| 状态/视角字号 | `10px` | `screenHeight * 0.012` |
| 放大图标尺寸 | `20px` | `screenHeight * 0.022` |

**每个监控画面布局**：

```
┌─────────────────────────────────────┐
│ 🟢 运行中            [□] 放大缩小  │ ← 左上：灰底圆点+状态；右上：图标
│                                     │
│         [    监控画面    ]          │
│                                     │
│                          监控视角   │ ← 右下：红底视角标签
└─────────────────────────────────────┘
```

### 2.5 状态卡片

## 3. 屏幕自适应规则

所有固定像素值改为基于 `screenHeight` 的比例计算：

```dart
final screenHeight = MediaQuery.of(context).size.height;
```

| 用途 | 比例 | 说明 |
|------|------|------|
| KPI 内边距 | `screenHeight * 0.012` | 约 7-10px |
| KPI 图标 | `screenHeight * 0.016` | 约 10-12px |
| KPI 标题/单位字号 | `screenHeight * 0.012` | 约 7-10px |
| KPI 数值字号 | `screenHeight * 0.024` | 约 14-18px |
| 监控画面高度 | `screenHeight * 0.16` | 约 95-130px |
| 监控图标 | `screenHeight * 0.04` | 约 24-32px |
| 状态字号 | `screenHeight * 0.012` | 约 7-10px |

## 5. 验证方法

1. 登录 → 选择奶厅 → 进入首页
2. 在不同尺寸模拟器上测试，确认布局自适应

4. 确认监控卡片高度自适应，状态/视角文字可读
5. 确认状态卡片等高，有图标
6. 确认无布局错误日志
