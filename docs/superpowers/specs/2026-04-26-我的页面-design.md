# 我的页面 - 设计规格文档

**日期**：2026-04-26
**页面**：我的页面（个人中心）
**类型**：业务功能 - 个人中心模块
**状态**：已确认

---

## 1. 页面概述

| 项目 | 内容 |
|-----|------|
| **页面定位** | 个人中心入口，展示用户信息和功能快捷入口 |
| **核心功能** | 用户信息展示、维保计划/记录/故障日志入口、系统设置、退出登录 |
| **前置条件** | 用户已登录 |
| **后置结果** | 可跳转至各功能页面或退出登录 |

---

```

│  ┌─────────────────────────────┐   │
```

### 区域划分

| 区域 | 内容 | 说明 |
|-----|------|-----|
| 导航栏 | "个人中心"标题 | 橙色背景，白色文字 |
| 用户信息 | 姓名+部门 + 个人信息入口 | 橙色背景，白色字体 |
| 业务入口分组 | 维保计划、维保记录、故障日志 | 白色卡片，圆角8px，无分组标题 |
| 系统入口分组 | 系统设置、检查更新、关于我们 | 白色卡片，圆角8px，无分组标题 |
| 退出按钮 | 退出登录 | 白色背景，橙色边框，橙色文字，居中 |
| 底部导航 | 4个Tab | "我的"页橙色高亮 |

## 3. 组件设计

### 3.1 导航栏

| 属性 | 设计规范 |

| **背景** | 橙色 #FF6600 |

| **状态栏** | 白色图标，与橙色背景对比 |

### 3.2 用户信息区

| 属性 | 设计规范 |
|-----|---------|
| **背景** | 橙色 #FF6600 |
| **字体** | 白色 |
| **左侧** | 用户姓名 + 部门，如"张三 - XX部门" |
| **右侧** | "个人信息" + 箭头▶ |

### 3.3 功能入口卡片

#### 分组结构

| 分组 | 入口项 |
|-----|------|

| 属性 | 设计规范 |

## 6. 数据结构

### 我的页面数据模型

```typescript
interface PersonalCenterPageData {
  // 用户信息
  user: {
    name: string;         // 用户姓名
    department: string;  // 部门
  };

  // 功能入口
  entries: {
    business: Array<{
      name: string;      // 维保计划/维保记录/故障日志
      icon: string;
      route: string;
    }>;
    system: Array<{
      name: string;      // 系统设置/检查更新/关于我们
      icon: string;
      route: string;
    }>;
  };
}
```
