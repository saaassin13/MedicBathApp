# 选择奶厅页面 - 设计规格文档

**日期**：2026-04-26
**页面**：选择奶厅页面
**类型**：认证流程
**状态**：已确认

---

## 1. 页面概述

| 项目 | 内容 |
|-----|------|
| **页面定位** | 用户选择要操作的奶厅/牧场 |
| **核心功能** | 搜索奶厅、展示奶厅列表、选中奶厅返回上一页 |
| **前置条件** | 用户已登录 |
| **后置结果** | 选中奶厅后返回上一页（若有），或跳转首页（若无上一页） |

## 3. 组件设计

### 3.1 导航栏

| 属性 | 设计规范 |
|-----|---------|
| **背景** | 橙色 #FF6600 |
| **标题** | "选择奶厅"，白色，居中，加粗18号 |
| **右侧按钮** | "X"关闭图标，白色，点击关闭页面（返回上一页，若无则跳转首页） |
| **左侧** | 无返回按钮 |

### 3.2 搜索框

| 属性 | 设计规范 |
|-----|---------|
| **样式** | 圆角矩形（24px），白色背景 |
| **边框** | 灰色边框，聚焦时橙色边框 |
| **图标** | 左侧放大镜图标，灰色 |
| **占位符** | "搜索奶厅"，灰色 |
| **宽度** | 占屏幕宽度，水平居中 |
| **内边距** | 水平16px |

### 3.4 奶厅列表

#### 列表项卡片

| 属性 | 设计规范 |
|-----|---------|
| **布局** | 水平排列：左侧图标 + 右侧文字 |
| **图标** | warehouse图标，浅橙色(#FFF3E0)圆角背景，橙色 |
| **文字格式** | "牧场名称-期数奶厅-编号转盘" |
| **示例** | "双城牧场-一期奶厅-一号转盘" |
| **卡片背景** | 白色 |
| **卡片圆角** | 12px |
| **间距** | 卡片间距12dp |

#### 选中状态

| 状态 | 样式 |
|-----|------|
| 未选中 | 白色背景，灰色边框(1px) |
| 选中 | 白色背景 + 橙色边框(2px) + 阴影 + 右侧勾图标 |

### 3.5 加载更多

| 属性 | 设计规范 |
|-----|---------|
| **文字** | "点击或下拉加载更多..." |
| **样式** | 居中显示，灰色小字 |
| **交互** | 支持点击或下拉刷新加载 |

## 5. 交互流程

```
┌──────────┐     ┌──────────┐     ┌──────────┐     ┌──────────┐
│ 进入页面 │ ──▶ │ (可选)   │ ──▶ │ 点击奶厅 │ ──▶ │ 返回上一页│
│          │     │ 搜索筛选 │     │   项     │     │ (或首页)  │
└──────────┘     └──────────┘     └────┬─────┘     └──────────┘
                                      │
                                      ▼
                               ┌──────────┐
                               │ 确认选中 │
                               │ 并返回  │
                               └──────────┘
```

### 交互细节

| 交互 | 行为 |
|-----|------|
| 点击搜索框 | 弹出键盘，开始输入筛选 |
| 点击列表项 | 选中该项，显示橙色边框 + 勾，返回上一页（若有），否则跳转首页 |
| 点击X关闭 | 关闭页面，返回上一页（若有），否则跳转首页 |
| 下拉列表 | 加载更多奶厅数据 |

## 6. 数据结构

### 奶厅数据模型

```
奶厅 {
  id: string,           // 唯一标识
  name: string,          // 显示名称
  farmName: string,      // 牧场名称
  period: string,       // 期数（如"一期"）
  number: string,        // 编号（如"一号转盘"）
  iconUrl: string       // 图标URL（可选）
}
```

### 搜索筛选

| 字段 | 说明 |

| keyword | 搜索关键字，匹配name字段 |
