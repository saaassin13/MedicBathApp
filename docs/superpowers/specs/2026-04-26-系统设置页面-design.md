## 5. 交互流程

```
┌────────────────────────────────────────────────────────┐
│                   进入系统设置页面                       │
└────────────────────────────────────────────────────────┘
                           │
                           ▼
┌────────────────────────────────────────────────────────┐
│                  查看和配置系统设置                      │
│                                                   │
│  ┌─────────────────────────────────┐              │
│  │ ❗ 异常告警推送        [开关]  │              │
│  ├─────────────────────────────────┤              │
│  │ 🗑 清除缓存                     │              │
│  └─────────────────────────────────┘              │
│                                                   │
└────────────────────────────────────────────────────────┘
                           │
            ┌──────────────┼──────────────┐
            ▼              ▼              ▼
      ┌──────────┐   ┌──────────┐   ┌──────────┐
      │ 切换开关 │   │ 清除缓存 │   │  返回    │
      │          │   │  弹出确认 │   │  点击X关闭│
      └──────────┘   └──────────┘   └──────────┘
```

### 交互细节

| 交互 | 行为 |
|-----|------|
| 点击开关 | 切换异常告警推送的开启/关闭状态 |
| 点击清除缓存 | 弹出确认对话框，用户确认后执行清除操作 |
| 点击X关闭 | 关闭页面，返回上一页 |

### 清除缓存确认流程

```
点击"清除缓存"
       ↓
   显示确认对话框
   ┌─────────────────┐
   │  清除缓存       │
   │  确定要清除缓存吗？│
   │                 │
   │  [取消]  [确定]  │
   └─────────────────┘
       ↓
  用户点击"取消" → 关闭对话框，不执行操作
  用户点击"确定" → 执行清除操作，显示成功提示
```
