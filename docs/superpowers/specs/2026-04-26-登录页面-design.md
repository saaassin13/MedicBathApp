# 登录页面 - 设计规格文档

**日期**：2026-04-26
**页面**：登录页面
**类型**：认证流程
**状态**：已确认

---

## 1. 页面概述

| 项目 | 内容 |
|-----|------|
| **页面定位** | 认证流程入口，用户接触系统的第一个界面 |
| **核心功能** | 账号密码输入、隐私政策确认、登录操作 |
| **前置条件** | 无（用户首次访问或登出后） |
| **后置结果** | 验证通过后进入选择奶厅页面 |

---

## 2. 布局结构

```
┌─────────────────────────────────────┐
│           [顶部品牌区]               │  ← Logo + 品牌标题
│                                     │
│           [账号输入框]               │  ← 用户图标 + 占位符
│                                     │
│           [密码输入框]               │  ← 锁头图标 + 显示/隐藏
│                                     │
│           [登录按钮]                 │  ← 橙色主按钮
│                                     │
│      [隐私政策勾选 + 法律文本]       │
└─────────────────────────────────────┘
```

### 区域划分

| 区域 | 内容 | 高度占比 |
|-----|------|---------|
| 品牌区 | Logo + "奶牛药浴大数据管理中心" | 约25% |
| 表单区 | 账号输入框 + 密码输入框 + 登录按钮 | 约50% |
| 底部区 | 隐私政策勾选 + 法律文本 | 约25% |

---

## 3. 组件设计

### 3.1 品牌区

| 元素 | 设计规范 |
|-----|---------|
| **Logo图标** | 橙色"S"形图形，居中显示 |
| **品牌文字** | "Scaling Robotics" 英文字符，橙色 |
| **业务标题** | "奶牛药浴大数据管理中心"，黑色加粗，适中字号 |

### 3.2 输入框

| 属性 | 设计规范 |
|-----|---------|
| **样式** | 大圆角矩形，16px圆角 |
| **背景** | 白色 |
| **阴影** | 轻微投影（Drop Shadow），增加立体感 |
| **左侧图标** | 功能图标（用户/锁头），灰色 |
| **占位符** | 灰色提示文字（"请输入账号"/"请输入密码"） |
| **右侧图标** | 密码框带显示/隐藏切换图标 |
| **高度** | 50-60dp，适合指尖操作 |
| **宽度** | 占屏幕宽度85-90%，居中 |
| **间距** | 两输入框间距16dp |

### 3.3 登录按钮

| 属性 | 设计规范 |
|-----|---------|
| **样式** | 圆角矩形，与输入框圆角一致 |
| **背景** | 橙色 #FF6600 |
| **文字** | 白色，适中字号，加粗 |
| **宽度** | 与输入框一致 |
| **高度** | 50dp |
| **交互** | 按下态（暗色遮罩）反馈 |

### 3.4 隐私政策勾选

| 属性 | 设计规范 |
|-----|---------|
| **勾选框** | 圆形单选框，灰色边框 |
| **文字** | "登录代表阅读并同意《隐私保护政策》和《用户协议》" |
| **链接色** | 橙色高亮（协议名称） |
| **间距** | 与登录按钮保持适当距离 |

## 5. 交互流程

```
┌──────────┐     ┌──────────┐     ┌──────────┐     ┌──────────┐     ┌──────────┐
│  进入页面 │ ──▶ │ 输入账号 │ ──▶ │ 输入密码 │ ──▶ │ 勾选政策 │ ──▶ │ 点击登录 │
└──────────┘     └──────────┘     └──────────┘     └──────────┘     └────┬─────┘
                                                                        │
                                                                        ▼
                                                             ┌──────────┐
                                                             │ 验证通过 │
                                                             └────┬─────┘
                                                                  │
                                                                  ▼
                                                         ┌──────────────┐
                                                         │ 进入选择奶厅 │
                                                         └──────────────┘
```

### 验证规则

| 字段 | 规则 |
|-----|------|
| 账号 | 非空，最小长度根据业务需求 |
| 密码 | 非空，最小长度根据业务需求 |
| 隐私政策 | 必须勾选 |

### 错误反馈

| 场景 | 反馈方式 |
|-----|---------|
| 账号为空 | 输入框下方红色提示文字 |
| 密码为空 | 输入框下方红色提示文字 |
| 未勾选政策 | 勾选框旁边红色提示文字 |
| 登录失败 | 弹窗提示错误原因 |

## 6. 系统适配

| 项目 | 设计要求 |
|-----|---------|
| **状态栏** | 透明沉浸式，与橙色标题栏融为一体 |
| **安全区** | 适配iOS刘海屏，底部预留系统指示条空间 |
| **屏幕适配** | 输入框宽度根据屏幕宽度自适应 |
| **暗色模式** | 预留扩展（当前版本不考虑） |

## 7. 验收标准

- [ ] 页面加载时品牌Logo和标题正确显示
- [ ] 账号/密码输入框可正常输入，左侧图标可见
- [ ] 密码输入框支持显示/隐藏切换
- [ ] 隐私政策链接可点击（跳转至协议页面）
- [ ] 登录按钮在填写完整前禁用或显示不可点击状态
- [ ] 点击登录后有加载状态反馈
- [ ] 验证失败时显示明确的错误提示
- [ ] 登录成功跳转至选择奶厅页面
- [ ] 页面适配各种屏幕尺寸和刘海屏
