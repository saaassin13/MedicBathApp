### Task 11: 屏幕自适应修复

**根因**：多处使用固定像素值（140px、24px、12px 等），在不同屏幕尺寸上表现不一致。

**修复方案**：使用 `MediaQuery.of(context).size.height` 作为基准，计算相对比例。

**MonitorCard 修改**：
```dart
final screenHeight = MediaQuery.of(context).size.height;

Container(
  height: screenHeight * 0.16,           // 替代固定 140
  child: Icon(size: screenHeight * 0.04),  // 替代固定 48
  child: Text(fontSize: screenHeight * 0.012),  // 替代固定 10-12
)
```

**KpiCard 修改**：
```dart
final screenHeight = MediaQuery.of(context).size.height;

Padding(padding: EdgeInsets.all(screenHeight * 0.012))  // 替代固定 12
Icon(size: screenHeight * 0.016)                         // 替代固定 16
Text(fontSize: screenHeight * 0.012)                     // 替代固定 12
Text(fontSize: screenHeight * 0.024)                     // 替代固定 24
```
